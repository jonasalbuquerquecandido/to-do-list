# 📝 ToDo List - JavaScript

Aplicação web simples para gerenciamento de tarefas (To-Do List), desenvolvida com **HTML, CSS e JavaScript puro (Vanilla JS)**.

O objetivo do projeto é praticar **manipulação do DOM, eventos e lógica de programação**, criando uma aplicação interativa que permite adicionar, concluir e remover tarefas.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## ⚙️ Funcionalidades

* ➕ Adicionar novas tarefas
* ✔️ Marcar tarefas como concluídas
* ❌ Remover tarefas da lista
* 📋 Atualização dinâmica da lista de tarefas
* 💾 Salvamento das tarefas no navegador usando **localStorage** 

---

## 📚 Conceitos praticados

Este projeto foi desenvolvido para treinar conceitos fundamentais de desenvolvimento front-end:

* Manipulação do **DOM**
* **Eventos** em JavaScript
* **Arrays e objetos**
* Estruturação de código
* Interatividade em aplicações web
* Persistência de dados no navegador *(localStorage)*
